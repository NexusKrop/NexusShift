# 贡献指南

首先，感谢你能参与贡献本项目。

如果你之前游玩过 GreenCreative、EnderShop 或者 RelaperStack，那么欢迎你回来。如果你发现之前的功能在这个项目里面没有，请创建一个议题。

## 报告议题

如果你想要报告议题的话，请遵循此处的指导。

### 漏洞

请确认你可以在 `main` 分支的最新版本复现你要报告的问题。如果你使用的不是最新版本，请从最新版本重新构建，然后尝试复现问题。

如果我们无法在最新版复现你的问题，你的议题会直接被标记 `状态::无效` 然后关闭。

如果你要为自己的代码贡献而创建议题，请待你的议题被标记 `状态::就绪` 后再创建合并请求。

### 功能请求

请确认你要请求的功能不在最新版本内。如果你想要对现有功能提出改进，请创建改进类议题。

请在此类议题内描述你期望的最终效果。

## 代码贡献

### 设置开发环境

在修改此项目之前，你需要设置开发环境。即使你认为你已经准备好开发环境，我们仍建议你根据此段比较。

#### 软件

你需要以下工具：

- OpenJDK 17（HotSpot JVM)
  - 我们建议使用 Adoptium 或者官方构建
- IntelliJ IDEA（社区版和终极版均可），最低 2021.3
  - 安装 Minecraft Development 插件
- 测试用服务端（可以直接在开发机上安装）
  - MC版本 1.18.1
  - 必须使用 Paper；Spigot 不好使
  - Windows 或者 GNU/Linux (M1 芯片的 Mac 和不带 GNU 的 Linux 发行版，以及 Linux/libre 内核不受支持）
  - CommandAPI（最新版即可）
- 游戏客户端

#### 如何安装

- 首先，准备好你的 Minecraft（启动器选择最新版本应该直接安装 1.18.1）
- 在[此处](https://www.jetbrains.com/idea/) 下载并安装 IDEA
- 在[此处](https://papermc.io/downloads) 下载并安装 PaperMC
- 下载并安装 [CommandAPI](https://commandapi.jorel.dev/) 直接作为插件丢进 `plugins` 即可。
- 克隆项目，开始工作！
